What is this?

This is a scratchpad for building custom formats for Microsoft Excel and Google Sheets.

Custom formats are confusing, but super powerful. We wanted to make them easier to create and understand.

This tool was built by your friends at Gridmaster.

Missing Features

This tool is under developement. Here are some things it's currently missing:

  • Support for format colors
  • Support for format conditions

If you notice any other issues or missing features, please let us know.
