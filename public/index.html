About

This is a scratchpad for building custom formats for Microsoft Excel and Google Sheets, built by your friends at Gridmaster.

Issues

This tool is in beta. Here are some things it's currently missing:

  • Support for format colors
  • Support for format conditions
  • Support for mixed or unreduced fractions
  • Alignment symbols (like ? and *) don't affect the alignment of the formatted cell.

If you notice any other issues or missing features, please report them here.
